&nbsp;Spendy - Aplicación de Gastos Hormiga



&nbsp;Arquitectura



Proyecto dividido en:



&nbsp;Backend

\- Spring Boot

\- Arquitectura MVC

\- Capas:

&nbsp; - Controller

&nbsp; - Repository

&nbsp; - Model



&nbsp;Frontend

\- React (Vite)

\- Componentes:

&nbsp; - FormularioGasto

&nbsp; - TablaGastos



&nbsp;Cómo ejecutar el Backend



1\. Entrar a la carpeta backend

2\. Ejecutar:

&nbsp;  mvn spring-boot:run

&nbsp;

&nbsp; Cómo ejecutar el Frontend



1\. Entrar a la carpeta frontend

2\. Ejecutar:

&nbsp;  npm install

&nbsp;  npm run dev
